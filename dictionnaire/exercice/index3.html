Exercise 4: Suivi des Employés

Vous devez gérer une liste d’employés avec les informations suivantes :

  • Nom
  • Poste
  • Salaire

Instructions :

  1. Créez une fonction initialiser_employes() qui retourne une liste de dictionnaires contenant trois employés avec leur poste et salaire.
    2. [
         {"nom": "Alice", "poste": "Développeur", "salaire": 4000},
         {"nom": "Bob", "poste": "Designer", "salaire": 3500},
         {"nom": "Charlie", "poste": "Chef de projet", "salaire": 5000}
     ]
  2. Créez une fonction ajouter_employe(employes, nom, poste, salaire) qui ajoute un nouvel employé à la liste.
  3. Créez une fonction augmenter_salaire(employes, nom, pourcentage) qui augmente le salaire d’un employé d’un certain pourcentage.
  4. Créez une fonction employe_salaire_max(employes) qui retourne le nom de l’employé avec le salaire le plus élevé.
