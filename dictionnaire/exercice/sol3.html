Exercice 4: Suivi des Employés

Question : Écrire un programme permettant de gérer une liste d’employés avec les informations suivantes :

  • Nom
  • Poste
  • Salaire

Instructions :

  1. Créer une fonction initialiser_employes() qui retourne une liste de dictionnaires contenant trois employés avec leur poste et salaire.
  2. Créer une fonction ajouter_employe(employes, nom, poste, salaire) qui ajoute un nouvel employé à la liste.
  3. Créer une fonction augmenter_salaire(employes, nom, pourcentage) qui augmente le salaire d’un employé d’un certain pourcentage.
  4. Créer une fonction employe_salaire_max(employes) qui retourne le nom de l’employé avec le salaire le plus élevé.
def initialiser_employes():
    return [
        {"nom": "Alice", "poste": "Développeur", "salaire": 4000},
        {"nom": "Bob", "poste": "Designer", "salaire": 3500},
        {"nom": "Charlie", "poste": "Chef de projet", "salaire": 5000}
    ]

def ajouter_employe(employes, nom, poste, salaire):
    employes.append({"nom": nom, "poste": poste, "salaire": salaire})

def augmenter_salaire(employes, nom, pourcentage):
    for employe in employes:
        if employe["nom"] == nom:
            employe["salaire"] += employe["salaire"] * (pourcentage / 100)

def employe_salaire_max(employes):
    max_salaire = 0
    employe_max = ""
    for employe in employes:
        if employe["salaire"] > max_salaire:
            max_salaire = employe["salaire"]
            employe_max = employe["nom"]
    return employe_max , max_salaire

Exercice 5: Gestion des Comptes Bancaires

Question : Écrire un programme pour gérer une liste de comptes bancaires contenant :

  • Nom du titulaire
  • Solde
  • Historique des transactions

Instructions :

  1. Créer une fonction initialiser_comptes() qui retourne une liste contenant trois comptes bancaires avec un solde initial et une liste vide pour l'historique des transactions.
  2. Créer une fonction effectuer_retrait(comptes, nom, montant) qui permet de retirer de l'argent d'un compte. Si le solde devient négatif après le retrait, la transaction doit être refusée.
  3. Créer une fonction effectuer_depot(comptes, nom, montant) qui permet de déposer de l'argent sur un compte. Le solde doit être mis à jour et l'historique des transactions doit être enregistré.
  4. Créer une fonction historique_transactions(comptes, nom) qui retourne l’historique des transactions d’un compte donné.

     def initialiser_comptes():
         return [
             {"titulaire": "Jean", "solde": 1500, "transactions": []},
             {"titulaire": "Sophie", "solde": 2500, "transactions": []},
             {"titulaire": "Marc", "solde": 500, "transactions": []}
         ]
     
     def effectuer_retrait(comptes, nom, montant):
         for compte in comptes:

                 if compte["solde"] >= montant:  # Vérifier si le solde est suffisant pour le retrait

                     print(f"Retrait de {montant} effectué.")
                 else:
                     print("Transaction refusée : solde insuffisant.")
