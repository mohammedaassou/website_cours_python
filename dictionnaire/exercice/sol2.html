Exercise 1: Gestion des stocks

Vous gérez un magasin et vous devez suivre les stocks des produits. Chaque produit a les informations suivantes :

  • Nom
  • Quantité en stock
  • Prix par unité

Instructions :

  1. Créez une fonction initialiser_stock() qui retourne un dictionnaire contenant les produits suivants :
    2. {
"Pommes": {"quantite": 100, "prix": 2},
"Bananes": {"quantite": 30, "prix": 3},
"Oranges": {"quantite": 50, "prix": 1.5}
}
  2. Créez une fonction ajouter_produit(stock, produit, quantite, prix) qui permet d'ajouter un nouveau produit ou d'ajouter à la quantité existante pour un produit existant.
  3. Créez une fonction valeur_totale_stock(stock) qui calcule la valeur totale du stock (quantité × prix pour chaque produit).

Correction:

# Fonction pour initialiser le stock

def initialiser_stock():
    stock = {
        "Pommes": {"quantite": 100, "prix": 2},
        "Bananes": {"quantite": 30, "prix": 3},
        "Oranges": {"quantite": 50, "prix": 1.5}
    }
    return stock

# Fonction pour ajouter un produit au stock

def ajouter_produit(stock, produit, quantite, prix):
        stock[produit] = {"quantite": quantite, "prix": prix}

# Fonction pour calculer la valeur totale du stock

def valeur_totale_stock(stock):
    total = 0
    for produit in stock:
        total += stock[produit]["quantite"] * stock[produit]["prix"]
    return total

Exercise 2: Analyse des étudiants

Vous avez une liste des notes sous la forme d'un dictionnaire. Chaque étudiant est associé à une liste de ses notes :

{
"Ali": [12, 15, 18],
"Sara": [16, 17, 16],
"Samir": [10, 9, 14]
}

Instructions :

  1. Créez une fonction calculer_moyenne(etudiants, nom) qui prend en entrée un dictionnaire d'étudiants et le nom d'un étudiant, et retourne sa moyenne.
  2. Créez une fonction meilleure_moyenne(etudiants) qui retourne le nom de l'étudiant avec la meilleure moyenne.

Correction:

# Fonction pour calculer la moyenne d'un étudiant

def calculer_moyenne(etudiants, nom):
        moyenne = sum(etudiants[nom]) / len(etudiants[nom])
        return moyenne
    

# Fonction pour trouver l'étudiant avec la meilleure moyenne

def meilleure_moyenne(etudiants):
    meilleur = ""
    meilleureNote = 0
    for nom, notes in etudiants.items():
        moyenne = sum(notes) / len(notes)
        if moyenne > meilleureNote:
            meilleureNote = moyenne
            meilleur = nom
    return meilleur , meilleureNote

Exercise 3: Gestion des commandes

Vous gérez une application de livraison. Les commandes des clients sont stockées dans un dictionnaire où chaque client a une liste de commandes :

{
"Client1": ["Pizza", "Salade"],
"Client2": ["Burger"],
"Client3": ["Sushi", "Soupe", "Salade"]
}

Instructions :

  1. Créez une fonction initialiser_commandes() qui retourne un dictionnaire contenant les commandes données ci-dessus.
  2. Créez une fonction ajouter_commande(commandes, client, commande) qui ajoute une commande à un client existant. Si le client n'existe pas, ajoutez-le au dictionnaire.
  3. Créez une fonction statistiques_commandes(commandes) qui retourne un dictionnaire contenant :
    • Le nombre total de commandes.
    • Le client ayant passé le plus de commandes.

Correction:

# Fonction pour initialiser les commandes

def initialiser_commandes():
    return {
        "Client1": ["Pizza", "Salade"],
        "Client2": ["Burger"],
        "Client3": ["Sushi", "Soupe", "Salade"]
    }

# Fonction pour ajouter une commande à un client

def ajouter_commande(commandes, client, commande):
    if client in commandes:
        commandes[client].append(commande)
    else:
        commandes[client] = [commande]

# Fonction pour obtenir les statistiques des commandes

def statistiques_commandes(commandes):
    total_commandes = 0
    client_max = ""
    max_commandes = 0
    
    for client, liste in commandes.items():
        nombre_commandes = len(liste)
        total_commandes += nombre_commandes
        if nombre_commandes > max_commandes:
            max_commandes = nombre_commandes
            client_max = client
    
    return {"total": total_commandes, "plus_de_commandes": client_max}
