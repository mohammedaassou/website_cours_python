Cours sur les Dictionnaires en Python

Introduction

Les dictionnaires en Python permettent de stocker des paires clé-valeur.

Création d'un Dictionnaire

mon_dictionnaire = {}
etudiant = {"nom": "Alice", "age": 22, "matiere": "Informatique"}
print(etudiant)
{'nom': 'Alice', 'age': 22, 'matiere': 'Informatique'}

Accéder aux Éléments

print(etudiant["nom"])  # Affiche "Alice"
print(etudiant.get("age"))  # Affiche 22
Alice
22

Modifier un Dictionnaire

etudiant["age"] = 23
etudiant["adresse"] = "Paris"
print(etudiant)
{'nom': 'Alice', 'age': 23, 'matiere': 'Informatique', 'adresse': 'Paris'}

Supprimer des Éléments

del etudiant["adresse"]
print(etudiant)
{'nom': 'Alice', 'age': 23, 'matiere': 'Informatique'}

Parcourir un Dictionnaire

for cle, valeur in etudiant.items():
    print(f"{cle}: {valeur}")
nom: Alice
age: 23
matiere: Informatique

Opérations Utiles

print(len(etudiant))  # Nombre d'éléments
print("nom" in etudiant)  # Vérifier une clé
etudiant.clear()  # Vider le dictionnaire
print(etudiant)
3
True
{}

Les Méthodes : items(), keys(), values()

1. items() - Retourne une liste de tuples contenant les paires clé-valeur

etudiant = {"nom": "Alice", "age": 22, "matiere": "Informatique"}
for cle, valeur in etudiant.items():
    print(cle, valeur)
nom Alice
age 22
matiere Informatique

items() permet d'itérer sur un dictionnaire en retournant chaque paire clé-valeur sous forme de tuple.

2. keys() - Retourne une vue des clés du dictionnaire

for cle in etudiant.keys():
    print(cle)
nom
age
matiere

keys() permet d'obtenir une vue des clés présentes dans le dictionnaire, idéale pour vérifier les clés existantes.

3. values() - Retourne une vue des valeurs du dictionnaire

for valeur in etudiant.values():
    print(valeur)
Alice
22
Informatique

values() permet d'obtenir une vue des valeurs présentes dans le dictionnaire, utile lorsque vous souhaitez manipuler uniquement les valeurs.

Exercices Pratiques

Exercice 1 : Informations sur une personne

Créez un dictionnaire nommé personne contenant les informations suivantes :

  • Nom : "Jean"
  • Âge : 30
  • Profession : "Développeur"

Ajoutez une clé "ville" avec la valeur "Paris". Ensuite, affichez toutes les informations de la personne en parcourant le dictionnaire.

personne = {
    "nom": "Jean",
    "age": 30,
    "profession": "Développeur"
}
personne["ville"] = "Paris"

for cle, valeur in personne.items():
    print(f"{cle}: {valeur}")
nom: Jean
age: 30
profession: Développeur
ville: Paris

Exercice 2 : Ajouter une couleur préférée

Créez un dictionnaire nommé infos contenant :

  • Nom : "Sophie"
  • Âge : 28

Ajoutez une clé "couleur" avec la valeur "Bleu", puis affichez le dictionnaire.

infos = {
    "nom": "Sophie",
    "age": 28
}
infos["couleur"] = "Bleu"

print(infos)
{'nom': 'Sophie', 'age': 28, 'couleur': 'Bleu'}

Conclusion

Les dictionnaires sont essentiels en Python pour manipuler des données efficacement. N'hésitez pas à expérimenter avec ces exercices !
